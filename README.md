# Python-FaceRecognition

## Insatllation

- Pull code from repo
  ```
  git pull https://github.com/vinbudii/Python-FaceRecognition.git
  ```
- Install Python
  ```
  https://www.python.org/downloads/
  ```
- Install Computer Vision
  ```
  pip install opencv-python
  ```
- Install Numpy
  ```
  pip install numpy
  ```
- Install Image
  ```
  pip install image
  ```
- Install DateTime
  ```
  pip install DateTime
  ```
- Install Tkinter
  ```
  pip install tk
  ```
